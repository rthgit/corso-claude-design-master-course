# Claude Design — Complete Master Course (English)

A full training path to master Claude Design from first use to expert/enterprise workflows.

## What This English Edition Includes

This English section contains:

- full 5-level learning path (L0 → L4)
- practical prompting frameworks
- hands-on exercises and level validation tests
- role-based workflows (Founder, PM, Marketing, Designer)
- enterprise layer (governance, handoff, limitations, workarounds)
- operational appendices (cheat sheets, checklists, glossary, FAQ)

---

## Course Curriculum

## Level 0 — Foundations

- What Claude Design is and is not
- Full interface map: chat, canvas, tweaks, web capture, export
- Product access and initial setup
- First complete cycle: prompt → iterate → export

## Level 1 — Beginner Execution

- Contextual prompting vs technical prompting
- P.A.T.C. framework (Product, Audience, Target, Character)
- 7 primary output categories (landing page, deck, app prototype, one-pager, social, wireframe, presentation)
- First iteration toolkit: chat, inline tweaks, direct text edits, adjustment knobs

## Level 2 — Intermediate Professional Use

- Automatic design system setup (codebase / brand docs / web capture)
- Team collaboration modes (private, internal link, edit access)
- Professional export strategy by use case
- Operational handoff toward implementation

## Level 3 — Advanced

- Advanced prompt engineering (layered brief, deliberate progression, anti-pattern prompting)
- Role-specific strategies (Founder, PM, Marketing, Designer)
- Iteration discipline and quality control checkpoints

## Level 4 — Expert / Enterprise

- Enterprise activation and design system governance
- Deep handoff bundle logic (design intent + tokens + component specs)
- Current limitations and practical workarounds
- Competitive context and strategic positioning

## Recommended Learning Workflow

1. Study levels in strict sequence (0 → 4).
2. Complete each practical exercise.
3. Pass each level test before moving forward.
4. Finish with one real end-to-end capstone project.

---

## Final Capstone Project

Goal: produce a full launch package in 2 hours:

- landing page
- sales deck
- 4 social creatives
- implementation handoff bundle

This capstone validates production-level course mastery.
