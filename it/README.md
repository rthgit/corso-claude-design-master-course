# Claude Design — Corso Completo (Italiano)

Percorso formativo completo per padroneggiare Claude Design dal livello base fino all'uso professionale/enterprise.

## Contenuto di Questa Versione

Questa sezione italiana contiene:

- metodologia completa in 5 livelli (L0 → L4)
- prompt framework applicabili subito
- esercizi guidati e test di verifica livello per livello
- workflow per ruoli (Founder, PM, Marketing, Designer)
- sezione enterprise (governance, handoff, limiti, workaround)
- appendici operative (cheat sheet, checklist, glossario, FAQ)

---

## Programma Didattico

## Livello 0 — Fondamenta

- Cos'è Claude Design e cosa non è
- Interfaccia completa: chat, canvas, tweaks, web capture, export
- Accesso al prodotto e setup iniziale
- Primo ciclo completo: prompt → iterazione → export

## Livello 1 — Beginner Operativo

- Prompting contestuale vs prompting tecnico
- Framework P.A.T.C. (Prodotto, Audience, Target, Carattere)
- 7 output principali (landing, deck, app prototype, one-pager, social, wireframe, presentazione)
- Prime iterazioni: chat, inline tweaks, edit diretto, adjustment knobs

## Livello 2 — Intermedio Professionale

- Design system automatico (codebase / brand docs / web capture)
- Collaborazione in team (privato, link interno, edit)
- Export professionale e casi d'uso per formato
- Handoff operativo verso sviluppo

## Livello 3 — Avanzato

- Prompt engineering avanzato (brief stratificato, progressione deliberata, anti-pattern)
- Strategie per ruolo (Founder, PM, Marketing, Designer)
- Regola pratica delle iterazioni e qualità dell'output

## Livello 4 — Expert / Enterprise

- Attivazione enterprise e governance del design system
- Handoff bundle in dettaglio (design intent + token + specs)
- Limitazioni attuali e workaround reali
- Confronto competitivo e visione strategica

## Workflow di Studio Consigliato

1. Studia in ordine i livelli 0 → 4.
2. Esegui gli esercizi pratici alla fine di ogni livello.
3. Verifica il test livello prima di passare al successivo.
4. A fine corso, esegui un progetto reale completo con output multipli.

---

## Progetto Finale (Capstone)

Obiettivo: produrre in 2 ore un pacchetto completo di lancio:

- landing page
- sales deck
- 4 creatività social
- handoff bundle per sviluppo

Questo capstone consolida tutte le competenze operative del corso.
