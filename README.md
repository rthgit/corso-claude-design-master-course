# Claude Design — Corso Completo / Complete Master Course

Benvenuto nella repository ufficiale del corso **Claude Design**.

Questo repository contiene un percorso completo, dalle basi fino al livello expert, con materiale in due lingue:

- 🇮🇹 Italiano: [Vai alla versione italiana](./it/README.md)
- 🇬🇧 English: [Go to English version](./en/README.md)

---

## Struttura Repository

- `it/README.md` → guida completa in italiano
- `en/README.md` → complete guide in English
- `corso_claude_design_expert.html` → corso completo in italiano (print-ready PDF)
- `corso_claude_design_expert_en.html` → complete English course (print-ready PDF)

---

## Obiettivo del Corso

Portare lo studente da **zero** a **expert operativo** su Claude Design, con:

- prompting framework professionali
- workflow end-to-end (brief → iterazione → export → handoff)
- best practice da fonti ufficiali + community
- output finali pronti per uso reale (PDF, PPTX, HTML, Canva, handoff)

---

## Come Scaricare in PDF

1. Apri uno dei file HTML del corso nel browser.
2. Usa `Ctrl+P` (o `Cmd+P` su Mac).
3. Seleziona **Save as PDF**.

---

## Nota Fonti

Il materiale combina:

- fonti ufficiali (Anthropic, Help Center Claude, Canva)
- fonti non ufficiali (community/dev: Reddit, GitHub)

La policy di studio consigliata è: **ufficiale per regole/capability, community per tattiche pratiche**.
